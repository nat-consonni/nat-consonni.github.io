  • Website Redesign for an E-Commerce Platform
    UX/UI Design - 2024

  • Improving Feature Discoverability in a SaaS Dashboard
    User Research - 2025

    DataMagician provides powerful analytics tools, but user feedback indicated that many essential features were underutilized due to poor discoverability. Our research found that hidden menus, lack of onboarding guidance, and an overwhelming interface led to confusion. To address this, we redesigned the dashboard layout with contextual tooltips, an improved search function, and a guided onboarding experience. The addition of an "Explore Features" section increased user adoption of key tools by 60%, while the updated navigation significantly reduced task completion times. These enhancements ensured that users could quickly access and leverage the platform’s full capabilities without frustration.
